# Computer Networks — Cisco Packet Tracer Labs

A collection of hands-on networking labs completed in **Cisco Packet Tracer**, covering core concepts in network design, protocols, and device configuration.

## About

This repository documents practical lab work done as part of a Computer Networks course. Each lab explores a specific networking concept through simulation and analysis in Cisco Packet Tracer (`.pkt` files).

## Topics Covered

- OSI Model layers and protocol simulation
- Ethernet LAN switching and ARP
- Switch and router configuration (console, CLI)
- Network topology design and troubleshooting

## Tools

- **Cisco Packet Tracer** — network simulation and analysis

## Notes

Each commit typically corresponds to a lab. See the [commit history](https://github.com/babek-aliyev/computer-networks/commits/main) for the full list of labs and their descriptions.
